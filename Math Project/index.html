Simpson's Rule Formula:

∫ f(x) dx ≈ (h / 3) × [f(a) + 4f(x1) + 2f(x2) + 4f(x3) + ... + 2f(xn-2) + 4f(xn-1) + f(b)]

Where:
- h is the width of each interval,
- n is the number of intervals (must be even),
- a is the lower limit,
